# Dark Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add dark mode to david-bloom.com with warm charcoal palette, OS preference default, localStorage toggle override, and a filled-circle toggle in the header.

**Architecture:** CSS custom properties already power the entire color system via `:root`. Dark mode adds a `[data-theme="dark"]` selector that overrides those variables. An inline script in `<head>` sets the attribute before paint (no FOUC). A `●` toggle in the nav flips the attribute and persists to localStorage.

**Tech Stack:** Astro, vanilla CSS, vanilla JS (no new dependencies)

**Spec:** `docs/superpowers/specs/2026-03-30-dark-mode-design.md`

### Task 2: Theme initialization script (no FOUC)

**Files:**
- Modify: `src/components/BaseHead.astro:49` (append at end of file)

- [ ] **Step 1: Add inline theme init script**

Append to the end of `BaseHead.astro`:

```html
<!-- Theme init (inline to prevent FOUC) -->
<script is:inline>
(function() {
	var saved = localStorage.getItem('theme');
	if (!saved) {
		saved = window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light';
	}
	document.documentElement.dataset.theme = saved;
})();
</script>
```

Note: `is:inline` is required — Astro's default script processing bundles and defers scripts, which would cause a flash. `is:inline` keeps it synchronous in `<head>`.

### Task 5: Dynamic accent color in AsciiCursorTrail

**Files:**
- Modify: `src/components/AsciiCursorTrail.astro:31` (the hardcoded `accentColor`) and surrounding script

- [ ] **Step 1: Replace hardcoded accent with dynamic reading**

In `AsciiCursorTrail.astro`, replace:

```typescript
const accentColor = [196, 97, 87]; // #C46157
```

with:

```typescript
function parseHexToRgb(hex: string): [number, number, number] {
  hex = hex.trim();
  if (hex.startsWith('#')) hex = hex.slice(1);
  return [
    parseInt(hex.slice(0, 2), 16),
    parseInt(hex.slice(2, 4), 16),
    parseInt(hex.slice(4, 6), 16),
  ];
}

function getAccentColor(): [number, number, number] {
  const val = getComputedStyle(document.documentElement).getPropertyValue('--accent');
  return parseHexToRgb(val || '#C46157');
}

let accentColor = getAccentColor();
```

- [ ] **Step 2: Add MutationObserver to re-read accent on theme change**

Add this block **inside the same `<script>` block**, right after the `accentColor` initialization (before the `document.addEventListener('mousemove', ...)` line):

```typescript
new MutationObserver(() => {
  accentColor = getAccentColor();
}).observe(document.documentElement, {
  attributes: true,
  attributeFilter: ['data-theme'],
});
```
